### schaeffer-bot

Meant to be used in Slack to return random Francis Schaeffer quotes.

##### Setup

```
$ git clone git@github.com:speric/schaeffer-bot.git
$ cd schaeffer-bot
$ rackup
$ curl -X POST http://localhost:9292/bot
```
